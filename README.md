# Practice Projects

This repository contains JavaScript practice projects built while learning frontend web development.

## Projects

### Form Validation

A simple login form that validates email and password inputs using JavaScript.

**Concepts Practiced**

* Form Validation
* Regular Expressions
* DOM Manipulation
* Event Handling

### Dynamic Profile Card Generator

A project that dynamically generates and updates profile cards based on user input.

**Concepts Practiced**

* DOM Manipulation
* Event Handling
* Dynamic Content Rendering
* JavaScript Fundamentals

## Technologies Used

* HTML5
* CSS3
* JavaScript
